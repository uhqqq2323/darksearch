Conditions d'utilisation
Dernière mise à jour : Mars 2024

1. Acceptation des conditions

En accédant à DarkSearcher, vous acceptez ces Conditions d'utilisation. Si vous n'acceptez pas, n'utilisez pas le service.

2. Description du service

DarkSearcher est une plateforme de recherche d'intelligence permettant de :

  • Rechercher dans notre base de données
  • Effectuer des lookups (Discord ID, Email, IP, Username)
  • Exporter et analyser les résultats

3. Conditions d'utilisation - Restrictions

Vous acceptez de :

  • Utiliser DarkSearcher uniquement pour des activités légales
  • Respecter les limites de requêtes de votre plan
  • Ne pas scraper, dupliquer ou vendre l'accès
  • Ne pas contourner les protections de sécurité

Interdictions absolues :

  • Harcèlement, doxxing ou ciblage malveillant
  • Utilisation pour discriminer ou opprimer
  • Publication des résultats sans consentement
  • Création de profils sans autorisation

4. Plans et facturation

Free : 10 requêtes/jour
Starter : 100 requêtes/jour
Pro : Illimité + API
Elite : Tout + Support 24/7

Les paiements via Discord Payments. Annulation sans frais possibles à tout moment.

5. Confidentialité et sécurité

E2EE : Toutes les recherches sont chiffrées. Aucune donnée enregistrée après votre session.
RGPD : Conformité complète. Voir la Politique de confidentialité.

6. Sources des données

Exclusivement des sources publiques légales. Aucune donnée obtenue illégalement ou sans consentement.

7. Limitation de responsabilité

DarkSearcher est fourni "TEL QUEL". Nous déclinons toute responsabilité pour :

  • Inexactitudes des résultats
  • Interruptions de service
  • Votre usage des données

8. Suspension et résiliation

Nous pouvons suspendre votre compte si vous :

  • Violez ces conditions
  • Utilisez le service de manière abusive
  • Engagez des activités illégales

9. Conformité légale

Vous acceptez de respecter toutes les lois applicables : RGPD, lois nationales, lois contre le harcèlement.

10. Contact

Questions ? discord.gg/rfshop ou admin@darksearcher.fr

Politique de confidentialité
Dernière mise à jour : Mars 2024

1. Introduction

Votre confidentialité est une priorité. Cette politique explique comment nous collectons, utilisons et protégeons vos données.

2. Données collectées

Compte

  • Nom d'utilisateur (chiffré)
  • Email (validé)
  • Mot de passe (hashé bcrypt)
  • Plan d'abonnement

Recherches

  • Vos requêtes ne sont PAS enregistrées
  • Seul le nombre de requêtes/jour est conservé
  • Aucun log des critères ou résultats

Sécurité

  • IP lors du login (24h max, puis suppression)
  • Tentatives de connexion échouées (prévention fraude)
  • Logs de sécurité (7 jours)

3. Utilisation des données

  • Authentification et accès au compte
  • Fourniture du service
  • Facturation et paiements
  • Détection des fraudes
  • Amélioration du service

4. Sécurité

TLS 1.3 : Toutes les communications chiffrées
Stockage : Données chiffrées au repos
Accès : Administrateurs autorisés uniquement

5. Partage de données

Nous ne partageons PAS vos données avec des tiers, sauf :

  • Si la loi l'exige (ordonnance judiciaire)
  • Vous l'autorisez explicitement
  • Pour protéger la sécurité publique

6. Cookies

Seul un cookie de session (expire à la fermeture). Pas de tracking, pas d'analytics tiers.

7. Rétention des données

  • Comptes : Tant que le compte existe
  • Suppression : 30 jours après suppression du compte
  • Logs sécurité : 7 jours
  • IPs : 24 heures
  • Paiements : 7 ans (légal)

8. Droits RGPD

Vous avez le droit de :

  • Accéder à vos données
  • Corriger les informations
  • Demander la suppression (droit à l'oubli)
  • Exporter vos données
  • Refuser le traitement

Contactez : admin@darksearcher.fr

9. Violation de données

En cas d'incident :

  • Notification dans 72h
  • Alertes des autorités si nécessaire
  • Détails complets fournis

10. Contact

Questions ? discord.gg/rfshop ou admin@darksearcher.fr

Questions fréquentes
Trouvez les réponses à vos questions
Pourquoi choisir DarkSearcher ? ▼
DarkSearcher est la plateforme d'intelligence la plus complète pour les professionnels. Nous offrons un accès à 50 millions de documents indexés, avec 4800+ sources actives mises à jour en temps réel. Notre infrastructure est conçue pour la vitesse (42ms en moyenne) et la sécurité absolue avec chiffrement end-to-end.
Comment fonctionne la recherche Document ? ▼
La recherche Document vous permet d'interroger notre base de données avec plusieurs critères : prénom, nom, téléphone, adresse, ville et département. Notre moteur combine ces critères pour trouver des correspondances exactes. Tous les résultats sont chiffrés et votre recherche est totalement anonyme.
Qu'est-ce que le Lookup ? ▼
Le Lookup permet de rechercher des informations sur la base d'identifiants uniques : Discord ID, Email, Adresse IP ou Username. Notre système utilise des APIs publiques et open-source pour vérifier et corroborer les informations trouvées de manière instantanée.
Combien de requêtes puis-je faire avec le plan Free ? ▼
Le plan Free vous donne accès à 10 requêtes par jour. C'est parfait pour explorer la plateforme et effectuer des recherches occasionnelles. Vous avez accès à la recherche Document complète et aux lookups basiques. Pour un usage intensif, nous recommandons les plans Starter, Pro ou Elite.
Quelle est la différence entre Starter et Pro ? ▼
Starter (10€/mois) : 100 requêtes/jour, recherche Document avancée, Lookup premium et support par email.

Pro (25€/mois) : Requêtes illimitées, tous les modules, accès API complet pour intégrer DarkSearcher à vos outils, et support prioritaire. Pro est la solution pour les professionnels qui utilisent intensivement la plateforme.
Comment fonctionne la facturation ? ▼
La facturation est simple et transparente. Vous êtes débité le jour où vous activez votre abonnement, puis chaque mois à la même date. Vous pouvez annuler à tout moment sans frais de résiliation. Pas d'engagement long terme, pas de surprise.
Mes données sont-elles sécurisées ? ▼
Absolument. Toutes vos recherches sont chiffrées de bout en bout (E2EE). Aucune données n'est stockée après votre session. Nous ne conservons pas vos requêtes, nous n'enregistrons pas vos résultats. Votre activité est 100% privée. Nous utilisons l'infrastructure Cloudflare pour la protection DDoS et la redondance.
Comment puis-je activer un plan payant ? ▼
Cliquez sur "Activer" sur la page Abonnements. Un modal s'ouvrira avec les instructions pour finaliser votre souscription. Le paiement se fait via notre serveur Discord officiel : discord.gg/rfshop. Contactez un administrateur qui validera votre achat immédiatement.
Puis-je changer de plan après inscription ? ▼
Oui, bien sûr ! Vous pouvez changer de plan à tout moment. Contactez simplement un administrateur sur Discord avec votre nouveau choix. Le changement prend effet immédiatement. Si vous passez à un plan supérieur, seule la différence est facturée. Pas de complication.
Comment puis-je bénéficier du support ? ▼
Le plan Free vous donne accès au support communautaire sur Discord. Les plans Starter+ bénéficient du support email prioritaire. Les plans Elite ont accès à un support 24/7 dédié et à un manager personnel. Rejoignez-nous sur discord.gg/rfshop.
Puis-je exporter mes résultats de recherche ? ▼
Les plans Free et Starter permettent l'export basic (CSV simple). Les plans Pro et Elite offrent l'export avancé avec exports illimités, formatage personnalisé (JSON, Excel, PDF) et automatisation via l'API. Parfait pour l'intégration dans vos workflows.
Comment sont mises à jour les données ? ▼
Notre base de données est mise à jour en temps réel. Nous agrégeons des milliers de sources publiques et les intégrons automatiquement grâce à nos crawlers propriétaires. 4800+ sources actives contribuent à nos 50 millions d'enregistrements. Les nouvelles données arrivent chaque jour.
DarkSearcher est-il légal ? ▼
DarkSearcher agrège exclusivement des données publiques provenant de sources officielles et légales. Nous respectons la RGPD et les lois sur la protection des données. Notre plateforme est conçue pour la conformité légale. Les utilisateurs sont responsables de l'usage qu'ils font de la plateforme conformément à leur juridiction locale.
Puis-je faire des recherches par API ? ▼
Oui ! Les plans Pro et Elite donnent accès à notre API REST complète. Vous pouvez automatiser vos recherches, les intégrer dans vos applications et construire des workflows personnalisés. La documentation API complète est disponible dans votre dashboard. Support API 24/7 inclus.
Y a-t-il une limite de temps de stockage pour les résultats ? ▼
Non. Une fois un résultat trouvé et stocké dans votre compte, il est conservé aussi longtemps que vous le souhaitez. Cependant, vos sessions de recherche ne sont pas enregistrées — seuls vos fichiers d'export personnels sont conservés. C'est optimal pour la sécurité et la confidentialité.
Puis-je downloader le client de bureau ? ▼
Actuellement, DarkSearcher fonctionne entièrement dans le navigateur web. Cette approche garantit que vous avez toujours la dernière version sans mise à jour nécessaire. Les clients de bureau et les applications mobiles sont en développement pour les plans Elite. Rejoignez Discord pour les nouvelles.
